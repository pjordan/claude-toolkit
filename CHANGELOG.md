# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.0] - 2025-01-18

### Added

- GitHub Actions CI workflow for automated validation
- Markdown linting with markdownlint-cli2
- Security checks for committed secrets and .env files
- Template server.py implementations for MCP server examples
- CI status badge in README

## [0.1.0] - 2025-01-18

### Added

- Initial repository structure with skills, subagents, and MCPs directories
- Skills framework with TEMPLATE.md and validation script
- Subagents framework with configuration templates
- MCP servers framework with validation script
- Example skills: code-review, data-analysis, a2a-agent
- Example subagents: research-assistant, code-generator
- Example MCP server templates: custom-api, database-connector
- Comprehensive documentation in docs/ directory
- Contributing guidelines
